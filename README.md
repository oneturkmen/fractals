## Fractls

Implementation of the ["Composing Fractals" by Mark P. Jones](http://web.cecs.pdx.edu/~mpj/pubs/composing-fractals.pdf).

Uses OpenGL Utility Kit (GLUT). Make sure it is installed.


Compile:

```bash
ghc -o fractals fractals.hs
```

Run:

```bash
./fractals
```
